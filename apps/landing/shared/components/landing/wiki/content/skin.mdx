# Система скинов

## Формирование скина

В зависимости от типа аккаунта игрок получает определённый скин:

- Лицензионный аккаунт – скин соответствует тому, который установлен в официальном профиле.  
- Пиратский аккаунт – по умолчанию используется скин Стива, но его можно изменить в игре.

## Как изменить скин

В игре изменить скин можно разными способами:  
указать ссылку на скин, либо указать никнейм игрока, скин которого вам нужен.

- Команда для изменения скина через никнейм:  
  ```bash
  /skin set [никнейм]

- Команда для изменения скина через ссылку:
  ```bash
  /skin url [ссылка] [тип скина]
